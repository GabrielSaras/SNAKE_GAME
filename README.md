🐍 Snake Game v2 (C)

Projeto acadêmico desenvolvido em linguagem C que implementa uma versão aprimorada do clássico jogo da cobrinha, com foco em interatividade, organização de código e aplicação de conceitos fundamentais de programação.

🎮 Funcionalidades
Movimento da cobra em tempo real
Detecção de colisão com paredes, corpo e obstáculos
Sistema de pontuação
Ranking persistente com armazenamento em arquivo
Menu interativo
Tela de pause
Tela de game over
🚀 Diferenciais do Projeto
👥 Modo 2 Jogadores

Permite que duas pessoas joguem simultaneamente no mesmo teclado, aumentando a interatividade do jogo.

🍎 Sistema de Frutas Customizáveis

O jogador pode escolher diferentes tipos de frutas, alterando o visual do jogo.

🎯 Sistema de Dificuldade
Fácil: sem espinhos
Normal: 4 espinhos fixos
Difícil: 8 espinhos com movimentação dinâmica
🎨 Uso de Emojis (UTF-8)

O jogo utiliza emojis para representar frutas, paredes e obstáculos, deixando a interface mais moderna e visualmente agradável.

🧠 Conceitos Aplicados
Estruturas de dados (arrays e structs)
Manipulação de arquivos (ranking)
Máquina de estados
Controle de entrada via teclado
Lógica de jogo (game loop)
Renderização em console
⚙️ Tecnologias Utilizadas
Linguagem C
Windows Console API
GCC (Dev-C++ / MinGW)
